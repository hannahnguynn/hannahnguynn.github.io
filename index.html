Projects

Hike Reservation

During quarantine, my fellow peers and I developed a program that replicated Disney's park reservation system. Instead, we applied it to National Parks in hopes of reducing days of overconcentration and stopping the spread. Users are able to choose make, view, and cancel their reservations, containing information regarding the date and time of their visit, their desired park, the size of their party.

RSA Decryption and Encryption

When users input their encryption key, a product of two prime numbers, and a message, the program implements Modular Expontiation to encrypt the given message. Inversely, when the user inputs an encrypted text file, the program implements both the basic and extended Euclidean Algorithm to decrypt the message into a separate text file, using the two prime command inputs, p and q.

Thermostat

Using a temperature sensor, Arduino microcontroller, LCD shield, buzzer, rotary encoder, buttons, and a servo motor, I created a thermostat to read and regulate the temperature of the room. The user is able to set tempurature bounds, and once the temperature goes above of below said bounds, the program will activate the air conditioning or heater accordingly. Additionally, the servo motor will display the current temperature; however, when the user is setting the bounds, the motor will display the current settings in sync with the rotary encoder input for 4 seconds and return to its inital setting.

Image Editor

The user is able to select, delete, and manipulate objects within the image processor through command inputs. Using breadth-first search (BFS), the program identifies and groups all objects in order to allow the user to move, cast, and edit their photos like the magic wand tool on Photoshop.

Movie Search

The Movie Search website imports and Apple API of movies to allow users to search up and discover the ratings, release dates, and more about their desired movie, in an aesthetic and simple-designed interface.
